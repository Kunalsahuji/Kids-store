# 🧸 Toyove India - Multi-Vendor Marketplace

**Toyove India** is a high-fidelity, mobile-first marketplace storefront designed for the children's toy and clothing industry. Inspired by the premium user experience of Etsy, this platform provides a seamless, touch-optimized discovery journey for parents and gift-seekers.

---

## 📌 Project Details (as per Appzeto - IT Services & Solutions)
- **Objective**: Rebuild the Toyove India storefront into a world-class, mobile-first marketplace.
- **Key Categories**: Targeted sections for Toys, Children's Clothing, Personalised Gifts, and Nursery Decor.
- **Design Philosophy**: Strict adherence to a **Mobile-First** architecture, ensuring pixel-perfect layout and interaction patterns on all mobile viewports.
- **Tech Stack**:
  - **Frontend**: React 19 + Vite
  - **Styling**: Tailwind CSS v4 (Modern Design System)
  - **Icons**: Lucide React + Custom SVG Icons
  - **Deployment**: Vercel

---

## ✨ Features
- **High-Fidelity Mobile UI**: Single-row high-density navigation and a premium marketplace aesthetic.
- **Dynamic Category Discovery**: Circular category "bubbles" and interactive portrait carousels.
- **Editorial Homepage Flow**: A curated sequence of Hero banners, Recently Viewed items, Shop Features, and Personalized Picks.
- **Interactive Marketplace Components**: 
  - Responsive product grids with wishlist/discount badges.
  - Glassmorphism hover effects for enhanced engagement.
  - Mobile-accordion footer for seamless link navigation.
- **Newsletter Integration**: Periwinkle blue promotional blocks with centered primary call-to-actions.
- **Performance Optimized**: Built with React 19 for industry-leading speed and responsiveness.
